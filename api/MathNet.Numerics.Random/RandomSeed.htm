Public Static Functions

int Guid()

Provides a seed based on time and unique GUIDs. WARNING: There is only low randomness in this seed, but at least quick repeated calls will result in different seed values. Do not use for cryptography!

int Robust()

Provides a seed based on an internal random number generator (crypto if available), time and unique GUIDs. WARNING: There is only medium randomness in this seed, but quick repeated calls will result in different seed values. Do not use for cryptography!

int Time()

Provides a time-dependent seed value, matching the default behavior of System.Random. WARNING: There is no randomness in this seed and quick repeated calls can cause the same seed value. Do not use for cryptography!
